# FAYM - Web Analytics Event Service

## Overview

A backend service to collect, store, and provide aggregated analytics for user interaction events (`view`, `click`, `location`).

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Python (Faker for data generation)
